## break

- 반복문 안에서 반복을 즉시 끝낼 때 사용
- 자신을 포함하는 반복문 하나만 벗어난다
- 반복문이 중첩된 경우 가장 안쪽에서 break를 사용하여 모든 반복문을 한 번에 벗어날 수 없음

> 안쪽 for문 하나만 탈출한다

```c
while()
{
    for()
    {
        if()break;
    }
}
```

> 반복문 블록 전체를 벗어난다!

```c
while ()
{
    ...
    if()
    {
        ...
        break;
    }

}
```

#### if문과 달리 switch ~ case문의 블록 안에서 break를 사용하면 switch ~ case 블록만 벗어난다
